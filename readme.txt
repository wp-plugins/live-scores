== Description ==

= Features =

The wordpress plugin 'Live Scores' is covering up to 7 sport events.

* Football
* Tennis
* Basketball
* Ice Hockey
* American Football
* Baseball
* Handball

= Easy as 1,2,3 =

To add the Live Scores to your website use one of the following short codes:


[livescores-red] - Red Layout

[livescores-green] - Green Layout

[livescores-white] - White Layout

[livescores-black] - Black Layout

[livescores-orange] - Orange Layout

[livescores-blue] - Blue Layout

[livescores-gray] - Gray Layout


= Is it 100% FREE? =

The .JS file which transmit the live scores data is provided by enetscores.com absolutely free of charge.

== Installation ==

1. Upload the `live-scores` folder to the `/wp-content/plugins/` directory
1. Activate the WordPress SEO plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `WP SEO Wizard` menu that appears in your admin menu
